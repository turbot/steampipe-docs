### STEAMPIPE_LOG_LEVEL
Sets the output logging level.  Standard log levels are supported (`TRACE`, `DEBUG`, `INFO`, `WARN`, `ERROR`). By default, the log level is `WARN`.

Logs are written to `~/.steampipe/logs/`

#### Usage 
```bash
export STEAMPIPE_LOG_LEVEL=TRACE
```
